# Визуализация алгоритмов поиска пути в лабиринте (BFS vs DFS)

Этот проект создан для наглядной демонстрации работы базовых алгоритмов обхода графов — поиска в ширину (BFS) и поиска в глубину (DFS). Проект подготовлен в рамках учебного доклада.

## Описание проекта
В репозитории представлены два варианта реализации визуализации на Python:

1. `random_maze.py` - генерирует небольшие случайные лабиринты.

2. `simple_map.py` - использует карту, созданную вручную.

## Ключевые особенности
* **BFS (Breadth-First Search):** Гарантирует нахождение кратчайшего пути. Использует структуру данных **Очередь** (FIFO).
* **DFS (Depth-First Search):** Не гарантирует кратчайший путь, но экономит память. Использует структуру данных **Стек** (LIFO).
